abstract class BeverageTemplate {
    public void prepareBeverage() {
        boilWater();
        addMainIngredient();
        stir();
    }

    abstract void addMainIngredient();

    abstract void stir();

    abstract void boilWater();
}

新設計有何優點?

這個程式用模板方法模式來做飲料製作。它把煮水、加主要材料、攪拌的流程寫在 prepareBeverage() 裡面，流程固定不變。不同飲料只要在自己的類別裡寫清楚怎麼煮水、加什麼材料、怎麼攪拌就行了，像 Tea 就是煮水、加茶葉、攪拌，Coffee 就是煮水、加咖啡粉、攪拌。這樣做的好處是流程統一、程式清楚，也方便新增其他飲料，只要新增類別寫自己的細節就好，不會改到原本的程式。主程式直接呼叫 prepareBeverage() 就能完成整個飲料製作，完全不用管每種飲料的細節，程式維護起來簡單，也很容易擴充。!@#$
